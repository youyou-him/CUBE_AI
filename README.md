## 프로젝트 개요 및 분석 목표

본 프로젝트는 브라질의 Olist E-Commerce Dataset (Kaggle)을 활용하여 고객 행동 데이터를 심층 분석하고, 확률론적 모델 기반의 CLTV (고객 생애 가치) 예측을 통해 미래 이탈 고객을 선제적으로 식별하고 맞춤형 마케팅 전략을 수립하는 것을 목표로 했습니다.

[cite\_start]최종적으로는 분석 결과를 직관적으로 이해하고 활용할 수 있도록 **CRM 분석 대시보드** 형태로 구현하여, 데이터 기반의 실질적인 비즈니스 의사결정을 지원했습니다.

## 분석 방법론 및 모델링 상세

### 1\. 고객 생애 가치 예측 (CLTV Modeling)

고객 생애 가치 예측을 위해 다음과 같은 확률론적 모델을 적용하여 고객의 미래 재구매 행동과 구매 금액을 분리하여 예측했습니다.

  * **BG/NBD 모델 (Beta-Geometric / Negative Binomial Distribution)**:
      * 개별 고객의 미래 재구매 횟수와 이탈 확률을 예측하는 데 사용됩니다.
      * 고객이 특정 기간 동안 얼마나 구매할지, 그리고 얼마나 오랫동안 활성 상태를 유지할지 확률적으로 모델링하여 재구매 주기 분석의 핵심 근거를 마련했습니다.
  * Gamma-Gamma 모델:
      * 고객이 거래 시 지출하는 평균 구매 금액을 예측하는 데 사용됩니다.
      * 이 모델은 고객의 재구매 빈도와 구매 금액이 서로 독립적이라는 가정 하에 작동하며, 최종 CLTV 금액을 산출하는 데 활용됩니다.

### 2\. 고객 이탈 예측 및 세그먼트 분류

  * **이탈 예측 모델:** 이탈 가능성 예측에는 **Gradient Boosting Machine (GBM)** 모델의 원리를 적용했습니다. GBM은 복잡한 고객 행동 피처들 사이의 비선형 관계를 학습하여 예측 정확도를 높이는 데 효과적입니다.
  * **세그먼트 분류:** CLTV 및 **RFM 분석** 결과를 통합하여 고객을 미래 가치와 행동 패턴에 따라 의미 있는 그룹으로 분류했습니다.
      * **가치 세분화:** 예측된 CLTV를 기준으로 'High Value'와 **'Low Value'** 그룹으로 나누어 마케팅 자원 투입의 우선순위를 설정했습니다.
      * **연령대/행동 세분화:** 'Elderly', 'Adult', 'Child', 'Teen' 등의 연령대 그룹으로 세분화하여, 각 그룹별 **재구매 유도율 및 이탈 민감도**를 분석했습니다.

## 기술 구현 및 서비스화

분석 결과를 활용한 실질적인 CRM 관리를 위해 웹 기반의 대시보드를 구축했습니다.

### 1\. 웹 및 시각화 스택 상세

| 구분 | 기술 | 구현 기능 및 특징 |
| :--- | :--- | :--- |
| **Frontend** | `HTML5`, `CSS`, `JavaScript` | 사용자 친화적인 대시보드 레이아웃 구현 및 UI 설계 |
| **CSS Framework** | **Tailwind CSS** | 반응형 웹 디자인 및 빠르고 일관된 모듈 기반 스타일링 적용  |
| **Chart Library** | **ApexCharts (JavaScript)** | CLTV, 이탈률, 세그먼트별 현황 등을 **동적으로 시각화**. 특히 **도넛 차트(Donut Chart)의 Total 값 기능**을 커스텀하여 세그먼트별 비율을 직관적으로 확인 가능하게 함. |
| **Template Engine** | `Thymeleaf` | 백엔드 데이터와의 동적 바인딩을 위한 HTML 템플릿 처리 |

### 2\. 구현된 대시보드 기능

#### 핵심 지표 시각화
- **ApexCharts 도넛 차트**로 고객 그룹 비율을 실시간 표시
  - High Value vs Low Value 분포 시각화
  - 연령대별(Elderly/Adult/Child/Teen) 세그먼트 비율
- **Total 값 커스텀 기능**으로 전체 고객 수 및 비율을 직관적으로 확인

#### CLTV 트렌드 모니터링
- **시계열 그래프**로 고객 가치 변화 추이 추적
- **이탈 확률 예측 결과**를 색상(Red/Yellow/Green)으로 구분
- 마케팅 담당자가 선제적으로 대응할 수 있도록 알림 기능 제공

#### 상세 고객 관리 페이지
- **개별 고객 프로필:** 구매 이력, 예측 CLTV, 이탈 점수
- **맞춤형 마케팅 액션 제안:** 재구매 유도, 이탈 방지, VIP 승격 전략 등
- **필터링 기능:** 연령대, 가치, 이탈 위험도 기준 고객 검색

## 향후 도전 과제

프로젝트의 실용성과 고도화를 위해 다음과 같은 향후 과제를 도출했습니다.

  * **리뷰 분석 고도화 (NLP):** 현재 정형 데이터에 한정된 분석을 넘어, 고객 리뷰 데이터(비정형 텍스트)를 분석하여 고객 경험을 심층 파악할 수 있는 **딥러닝 NLP 모델 (BERT 모델)** 도입 검토.
  * **분석 운영 자동화 (AutoML):** 데이터 전처리 및 모델 재학습 프로세스를 **AutoML** 기반으로 자동화하여, 실제 운영 환경에서의 모델 관리 및 효율성을 극대화하는 방안 연구.
  * **Backend & DB 연동:** 현재의 프론트엔드 대시보드를 실제 백엔드 서버 (Flask/Spring Boot)에 연동하여 데이터베이스(MySQL, PostgreSQL)와 실시간 통신하며 분석 결과를 제공하는 시스템 구축을 목표로 합니다.
